Cách chạy:
Vào folder chứa file .exe của chương trình, nhấn Shift + chuột phải, chọn "Open in Terminal"
Chọn file svg muốn parse, Shift + chuột phải, chọn "Copy as path"
Trong Terminal ghi .\SVG-Reader (hoặc tên của file .exe) và path của file svg
Example: .\SVG-Reader "D:\SVG-Reader-main\SVG-Reader\x64\Debug\apple.svg"
